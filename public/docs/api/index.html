Базові команди


- `npm install` — встановлення залежностей
- `npm run dev` — запуск локального сервера
- `npm run docs` — генерація документації (JSDoc)
- `npm run storybook` — запуск Storybook
- `npx storybook@latest init` — ініціалізація Storybook (в процесі)

Лабораторна №2

Додано основну логіку гри через кастомний хук:

  • хук useGameState() зі станами:
    • поле 6×7
    • поточний гравець
    • Поява фішки
    • перевірка переможця
    • перевірка нічиєї
    • скидання гри
  • GamePage отримує стан гри та передає результат у ResultPage
  • ResultPage відображає переможця або нічию

Лабораторна №3

  • Додано налаштування гри та завершення раундів через модальне вікно:

  • створено форму налаштувань на react-hook-form + yup

  • налаштування зберігаються у GameSettingsContext та localStorage

  • параметри, які можна змінювати:

  • кількість фішок для перемоги (winLength)

  • кількість перемог для виграшу матчу (maxWins)

  • колір гравця 1 та гравця 2

  • у useGameState() використовується winLength для перевірки перемоги

  • у GamePage ведеться підрахунок перемог за раунди з урахуванням maxWins

  • реалізовано модальне вікно завершення гри через портал (Modal + createPortal), яке:

  • показує переможця раунду або нічию

  • відображає поточний рахунок гравців

  • у разі досягнення maxWins показує фінального переможця матчу

    • надає можливість:
    • розпочати наступний тур
    • повернутися в головне меню

Лабораторна №4

Реалізовано роутинг та стилізацію застосунку:

Роутинг

  • додано react-router-dom

  • створено динамічні маршрути з використанням :userId:

    • /user/:userId/start
    • /user/:userId/game

  • редірект з кореня / → /user/guest/start

  • навігація реалізована через useNavigate()

  • useParams() використовується для отримання userId на сторінках

Стилізація

  • сторінку StartPage та GamePage переведено на CSS Modules
    • StartPage → StartPage.module.css
    • GamePage → GamePage.module.css
    • імпорт стилів через import styles from "./... .module.css"
  • інші сторінки стилізовані через стандартні CSS-файли
